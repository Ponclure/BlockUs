## Proposing Contributions

At Ponclure, we take contributions to a level of seriousness when accepting. We require that contributors should meet the following requirements before
contributing to the plugin project.

- Skilled Level of Java
- Familier with Spigot API and/or Paper API
- Made Plugins Before
- Knows How to Avoid Common Mistakes, such as:
  - Static Abuse
  - DRY Principles
  - Lack of OOP Understanding
  - Poor Code Design/Structure
  - etc...

We recommend you follow these guidelines first before contributing to the project. This is to ensure our project is maintained correctly and neatly.
